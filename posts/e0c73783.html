Investment01

1. 移动平均线 (MA)

  • 定义：计算一段时间内的平均价格，平滑价格波动，反映趋势方向。
  • 常见类型：
    • 简单移动平均线 (SMA)：如5日、10日、20日、50日。
    • 指数移动平均线 (EMA)：更重视近期价格，反应更快。
  • 使用方法：
    • 金叉：短期均线（如5日）上穿长期均线（如20日），看涨信号。
    • 死叉：短期均线下穿长期均线，看跌信号。
    • 支撑/阻力：均线常作为动态支撑或阻力位。
  • 应用：判断趋势方向，适合中长期交易。

2. 相对强弱指数 (RSI)

  • 定义：衡量价格超买或超卖的震荡指标，范围0-100。
  • 计算：RSI = 100 - [100 / (1 + 平均上涨幅度/平均下跌幅度)]，通常取14日。
  • 使用方法：
    • 超买：RSI > 70，可能回调，考虑止盈。
    • 超卖：RSI < 30，可能反弹，考虑买入。
    • 背离：价格创新高而RSI未创新高，暗示反转。
  • 应用：捕捉短期买卖点，适合震荡行情。

3. 随机震荡指标 (Stochastic Oscillator, KDJ)

  • 定义：比较收盘价与一段时间内价格范围的关系，判断超买超卖。
  • 组成：%K（快线）、%D（慢线），范围0-100。
  • 使用方法：
    • 超买：%K > 80，可能下跌。
    • 超卖：%K < 20，可能上涨。
    • 交叉信号：%K上穿%D为买入信号，反之为卖出信号。
  • 应用：短线交易，适合波动市场。

4. 布林带 (Bollinger Bands)

  • 定义：由中轨（通常20日SMA）和上下轨（中轨±2倍标准差）组成，反映价格波动范围。
  • 使用方法：
    • 突破上下轨：价格突破上轨看涨，下轨看跌，但需确认趋势。
    • 收窄：布林带收窄预示大行情将至。
    • 反弹：价格触碰上下轨后常回撤至中轨。
  • 应用：判断趋势强度和反转点，适合波段交易。

5. MACD (指数平滑异同移动平均线)

  • 定义：通过快线（DIF）、慢线（DEA）和柱状图（MACD）分析趋势和动能。
  • 计算：
    • DIF = 12日EMA - 26日EMA
    • DEA = DIF的9日EMA
    • MACD柱 = DIF - DEA
  • 使用方法：
    • 金叉：DIF上穿DEA，买入信号。
    • 死叉：DIF下穿DEA，卖出信号。
    • 柱状图：柱由负转正看涨，由正转负看跌。
  • 应用：捕捉趋势转折，适合中短期交易。

6. 平均真实波幅 (ATR)

  • 定义：衡量价格波动幅度，常用于设定止盈止损。
  • 计算：取一段时间内真实波幅（TR）的平均值，TR = Max(高-低, |高-前收|, |低-前收|)。
  • 使用方法：
    • 止损：买入价 - 2×ATR。
    • 止盈：买入价 + 3×ATR。
  • 应用：动态调整交易范围，适合波动性大的股票。

7. 成交量 (Volume)

  • 定义：反映市场参与度和资金流动，需结合K线分析。
  • 使用方法：
    • 放量上涨：看涨，趋势可能持续。
    • 缩量下跌：可能反弹，卖压减弱。
    • 异常放量：结合K线形态判断反转或突破。
  • 应用：验证K线信号的可靠性。

8. 斐波那契回调 (Fibonacci Retracement)

  • 定义：利用斐波那契比例（23.6%、38.2%、50%、61.8%）预测支撑和阻力位。
  • 使用方法：
    • 从波段低点拉到高点，回调至38.2%或50%可能是买入点。
    • 从高点拉到低点，反弹至61.8%可能是卖出点。
  • 应用：确定关键价格区间，适合波段交易。

9. 一目均衡表 (Ichimoku Cloud)

  • 定义：综合指标，包括转换线、基准线、云带等，反映趋势、支撑阻力。
  • 使用方法：
    • 价格在云带上方：看涨。
    • 价格在云带下方：看跌。
    • 云带颜色变化：绿色看涨，红色看跌。
  • 应用：全面判断趋势，适合中长期分析。

10. 支撑位与阻力位

  • 定义：基于历史K线高低点，判断价格可能停顿或反转的区域。
  • 使用方法：
    • 支撑位：历史低点或密集成交区，跌至此处可能反弹。
    • 阻力位：历史高点或密集成交区，涨至此处可能回落。
  • 应用：结合K线形态设定止盈止损。

如何结合K线使用技术指标

  1. 确认趋势：
    • 用MA或MACD判断大方向，避免逆势操作。
  2. 寻找买卖点：
    • RSI、KDJ、布林带适合捕捉超买超卖信号。
  3. 验证信号：
    • 成交量和K线形态（如锤头线、吞没形态）确认指标信号。
  4. 设定止盈止损：
    • ATR和支撑阻力位提供具体价格参考。

示例

假设当日K线：开盘100元，收盘103元，最高105元，最低98元。 - MA：5日均线102元，价格突破看涨。 - RSI：若为75，接近超买，考虑止盈。 - 布林带：上轨106元，接近时可止盈；下轨97元，跌破可止损。 - MACD：DIF上穿DEA，买入信号。

注意事项

  • 指标局限：单一指标可能失灵，建议组合使用（如MACD+RSI）。
  • 时间周期：短线用1分钟/5分钟K线，长线用日K线或周K线。
  • 市场环境：震荡行情用RSI/KDJ，趋势行情用MA/MACD。
